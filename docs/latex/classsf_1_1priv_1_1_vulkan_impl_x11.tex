Tell whether or not the system supports \doxylink{classsf_1_1_vulkan}{Vulkan}. 

This function should always be called before using the \doxylink{classsf_1_1_vulkan}{Vulkan} features. If it returns false, then any attempt to use \doxylink{classsf_1_1_vulkan}{Vulkan} will fail.

If only compute is required, set {\itshape require\+Graphics} to false to skip checking for the extensions necessary for graphics rendering.
